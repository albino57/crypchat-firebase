--- INSTALAR AMBIENTE E PROJETO FIREBASE ---

1- Instalar o Node.js ( https://nodejs.org/ ):
   - Depois da instalação, abra seu terminal e digite `node -v` e `npm -v` para checar.

2- Instalar as ferramentas do Firebase globalmente no computador:
   	npm install -g firebase-tools

3- Fazer login na conta Google através do terminal:
   	firebase login

4- Atualize o npm:
	npm install -g npm@latest


5- Criar o projeto frontend (React + Vite):
   	npm create vite@latest crypchat-firebase -- --template react

6- Instalar as dependências do projeto:
	npm install

7- Instalar as dependências do Frontend (Tailwind, etc.):

   npm install -D tailwindcss postcss autoprefixer
   npx tailwindcss init -p
	OBS: npx tailwindcss init -p causou problemas no meu ambiente e a solução foi usar npm install -D tailwindcss@3 e 	depois npx tailwindcss init como segue no site https://v3.tailwindcss.com/docs/installation.

   npm install react-router-dom crypto-js

   npm install react-textarea-autosize

8- Iniciar o Firebase dentro do projeto (o passo mais importante!):
   firebase init

   - O terminal vai te fazer algumas perguntas:
     - "Are you ready to proceed?" -> Yes
     - "Which Firebase features do you want to set up?" -> Use a barra de espaço para selecionar:
       - Firestore
       - Functions
       - Hosting
     - Depois, ele vai pedir para conectar a um projeto Firebase (você pode criar um novo).
     - Ele vai perguntar sobre as configurações de cada serviço (pode usar as opções padrão por enquanto).
     - Para Functions, escolha "TypeScript".
     - Para Hosting, ele perguntará qual o seu diretório público. Digite "dist".
     - Configurar como um single-page app? -> Yes

9- Instalar outras dependências:
	9.1- No diretório functions do projeto instale:
		npm install firebase-admin@latest

	9.2 Na raiz do projeto instale:
		npm install react-firebase-hooks

	9.3 Na raiz do projeto instale:
		npm install firebase-admin
	
	9.4 Na raiz do projeto instale:
		npm install firebase-functions
